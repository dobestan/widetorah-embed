6 Widget Types

Drop any of these into your HTML. Customize with data-* attributes.

Verse Card

Show any specific verse with translation badge and copy button.

<div data-widetorah="verse"
     data-ref="Genesis 1:1"
     data-theme="light"></div>

Verse of the Day

Auto-refreshes daily. Cached in localStorage for performance.

<div data-widetorah="votd"
     data-theme="dark"></div>

Chapter Card

Chapter summary with verse count and link to full text.

<div data-widetorah="chapter"
     data-ref="Genesis 1"
     data-theme="sepia"></div>

Comparison Card

Side-by-side verse comparison with link to compare page.

<div data-widetorah="compare"
     data-a="Genesis 1:1"
     data-b="Deuteronomy 6:4"></div>

Search Box

Redirects to WideTorah search. Great for sidebars.

<div data-widetorah="search"
     data-placeholder="Search Torah…"></div>

Compact Verse of the Day

Smaller layout for sidebars, footers, or tight spaces.

<div data-widetorah="votd"
     data-size="compact"
     data-theme="sepia"></div>

Installation

Three ways to embed — choose what fits your stack.

⚡

jsDelivr CDN

Recommended

Global CDN, auto-updates when new versions publish to npm. Fastest option.

<script src="https://cdn.jsdelivr.net/npm/
widetorah-embed@1/dist/embed.min.js"></script>
☁️

R2 CDN

Hosted directly by WideTorah. Served from Cloudflare R2 with immutable caching.

<script src="https://cdn.widetorah.com
/embed.min.js"></script>
📦

npm Package

Use with Webpack, Vite, Rollup, or any bundler. ESM and UMD formats included.

npm install widetorah-embed

// In your JS/TS file:
import 'widetorah-embed';

API Reference

All data-* attributes supported by WideTorah embed widgets.

Attribute Values Default Description
data-widetorah verse, chapter, person, compare, votd, search required Widget type to render
data-ref "Genesis 1:1" required for verse/chapter Torah reference (book chapter:verse)
data-slug "moses", "paul" required for person Person slug (person widget only)
data-a, data-b verse refs required for compare Two verse refs for comparison
data-theme light, dark, sepia light Visual theme
data-size default, compact default Widget size variant
data-translation jps, etc. jps Translation code
data-show-original true, false false Show original language text below verse
data-placeholder any string "Search Torah…" Search box placeholder text
data-lang en, ko, ja, es, … en UI language for widget labels
